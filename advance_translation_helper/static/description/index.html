Translate base odoo file '.po' with the help of Google Translate.

For this module to translate the base odoo file '.po' with the help of Google Translate, in all necessary languages.

  • Translate the content of a .po file by selecting the language to translate.
  • Load the translation and replace it with an existing term.

Requirement: pip3 install googletrans==3.1.0a0
