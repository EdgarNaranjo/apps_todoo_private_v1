New Purchase Order: For all odoo or only for sales order lines.

With the module the user could:

  • Correctly configure the product.
  • Set the route "MTO".
  • Establish if we want a purchase order to be generated from sales.
  • Establish if we want an order to be generated in all purchasing processes.

The workflow is as follows:


A product of type "Storable" is configured.



Configure the Purchases section of the product.



"MTO" and "Buy" routes are established.



It is configured when we want a new purchase order to be generated.



If we select the "Always" option wherever a purchase order is created, a new order will be generated.



If we select the "Only sale line" option it will only create a new order when it is generated from a sales order.
